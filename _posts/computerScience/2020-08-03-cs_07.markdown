># 프로세스와 스케줄러 이해 - 프로세스 구조  

패스트캠퍼스 강의 보면서 정리하고 있습니다  


>### 프로세스 구성  

- stack : 임시 데이터(함수 호출, 로컬 변수 등)  
- heap : 코드에서 동적으로 만들어지는 데이터  
- data : 변수/초기화된 데이터  
  - BSS :  초기화되지 않은 전역 변수  
  - DATA : 초기 값이 있는 전역 변수  
- text(CODE) : 코드
