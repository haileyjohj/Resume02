>## 기본형 매개변수, 참조형 매개변수

자바에서 메서드를 호출할 때 매개변수로 지정한 값을 메서드의 매개변수에 복사해서 넘겨줌.


### 기본형(primitive type) 매개변수

변수의 값 읽기만 가능

### 참조형(reference type) 매개변수

저장된 곳의 주소를 알 수 있기 때문에 변수의 값을 읽고 변경 가능
